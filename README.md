# dotbim-ifc

## Purpose

Converts to and from IFC and dotbim.

## Prerequirements

You need to install dotbimpy library (https://github.com/paireks/dotbimpy):

Tested with Python38

```cmd
pip install dotbimpy
pip install ifcopenshell
pip install PyQt5
```

Open: run.bat
